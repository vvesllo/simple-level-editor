# Simple Level editor

Simple level to text file editor

![](preview.png)


# Help:
+ F1 - previous image
+ F2 - next image
+ TAB - grid opacity
+ CTRL + S - save *\*.glf/\*.gdf*
+ CTRL + O - open *\*.glf/\*.gdf*
+ LMB - place tile
+ LMB - remove tile
> + lcf - Level Colliders File
> + ldf - Level Decorations File
